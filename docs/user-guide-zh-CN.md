# 财务及仓储管理系统使用说明

适用对象：不熟悉电脑和网页操作的使用人员

这份说明尽量用最简单的话来写。遇到不懂的地方，不要着急，可以按顺序一步一步看。

## 一、先记住这几件事

- 进入系统后，左边是菜单，点一下菜单名称，就会切换到对应页面。
- 页面里看到“保存”“确认登记”“确认保存”这类按钮时，点一次后请稍等，不要连续多点。
- 带有“*加载中...*”“*保存中...*”“*处理中...*”字样时，说明系统正在工作，请等待完成。
- 如果看到弹出的提示框，请认真看提示内容，再决定是否继续。
- 不确定时，先不要删除数据。删除后有些内容不能直接恢复。

## 二、登录系统

### 1. 已有账号，怎么登录

1. 打开系统登录页面。
2. 在“用户名”里输入自己的用户名。
3. 在“密码”里输入自己的密码。
4. 点“登录”按钮。
5. 登录成功后，会进入系统首页。

### 2. 没有账号，怎么申请

1. 在登录页点“注册”。
2. 输入想使用的用户名和密码。
3. 点“提交注册申请”。
4. 等管理员审核通过后，才可以登录。

### 3. 登录后怎么看自己是谁

- 页面左侧上方会显示当前登录人的用户名。
- 如果名字后面带“超级管理员”或“管理员”，说明这个账号有更多管理权限。

### 4. 退出系统

- 点左边菜单最下面的“退出登录”。

## 三、首页怎么用

首页名称是“概览仪表盘”。

这里主要用来看大概情况，不是主要录入页面。

你可以在这里看到：

- 今日交易笔数
- 最近流水
- 管理员还能看到应收、应付等金额

管理员在首页右上角还能看到：

- “备份与恢复”
- “导出 Excel”

如果你只是日常记账人员，看不到这些按钮是正常的。

## 四、最常用功能：记账登记

左边点“记账登记”。

这是系统里最常用的页面。销售、采购、收款、付款，大多都在这里登记。

### 1. 记账前先选“类型”

常见类型说明：

- 出库：卖货、发货
- 入库：进货、采购入库
- 销售退货：客户把货退回来
- 采购退货：把货退给供应商
- 收款：收到客户的钱
- 付款：付给供应商的钱
- 收款差额：收款时有零头差异
- 付款差额：付款时有零头差异

如果不知道选哪个，最简单的判断方法是：

- 卖出去，用“出库”
- 买进来，用“入库”
- 收到钱，用“收款”
- 付出去钱，用“付款”

### 2. 记账时一般按这个顺序填写

1. 选交易类型。
2. 选交易对象，也就是客户或供应商。
3. 填交易日期。
4. 如果页面有单据簿号和单号，就按实际单据填写。
5. 按页面要求填写产品明细，或者填写物料/加工明细。
6. 确认金额是否正确。
7. 需要的话，在“备注”里补充说明。
8. 点“确认登记”。

### 3. 出库、销售退货时怎么填

- 先选客户。
- 再选产品。
- 填数量和单价。
- 系统会自动算小计和金额。

如果没有找到要选的产品：

- 可以点“+ 临时新增产品”
- 先把产品名称、单位等信息补进去
- 保存后再继续登记

### 4. 入库、采购退货时怎么填

- 先选供应商。
- 再填写物料或加工明细。
- 一行不够可以点“+ 添加明细行”继续加。
- 没有对应物料或加工项时，可以临时新增。

### 5. 收款、付款时怎么填

- 先选客户或供应商。
- 再填金额。
- 如果有对应单据，可以在备注里写清楚。

### 6. 页面下方“最近记账”有什么用

这个地方可以看到最近录入的记录。

你可以在这里做这些事：

- 查看刚才是否登记成功
- 按不同方式排序
- 删除刚录错的记录
- 临时清空“最近记账”的显示

注意：

- “一键清空最近记账”只是把这个页面的显示先清掉，不是删除系统全部数据。
- 如果想恢复显示，可以点“恢复显示”。

## 五、客户与欠款怎么用

左边点“客户与欠款”。

这里用来管理客户、供应商，以及查看每个对象的流水和往来余额。

### 1. 新增客户或供应商

1. 点“+ 新增客户”。
2. 填名称。
3. 选择“客户”或“供应商”。
4. 点“保存”。

简单理解：

- 客户：买我们货的人
- 供应商：卖给我们货的人

### 2. 查找客户或供应商

- 在搜索框输入名称。
- 页面会自动筛选出匹配结果。

### 3. 看某个客户的流水

1. 在列表里找到对应的人。
2. 点“查看流水”。
3. 进入后就能看到这个对象的全部流水明细。

在流水详情页面，你还可以：

- 看联系人、电话、地址、备注
- 修改基础资料
- 按日期筛选流水
- 按类型筛选流水
- 按簿号筛选流水
- 按记账人筛选流水
- 搜索单号、交易明细、备注

### 4. 欠款怎么看

在客户列表里：

- 如果显示“对方欠我们”，说明客户还没把钱付清
- 如果显示“我们欠对方”，说明我们还没把钱付清
- 如果显示“两清”，说明目前结清了

### 5. 删除流水或补填单号

在流水详情里，每一条记录后面有操作按钮。

可以做的事情有：

- “补填/修改单号”
- “删除”

删除前请先确认，不确定就不要删。

### 6. 导出流水

如果你的账号有权限，会看到导出按钮。

可以导出：

- 当前对象流水
- 总流水

如果没有看到导出按钮，说明当前账号没有导出权限。

## 六、库存管理怎么用

左边点“库存管理”。

这里主要用于看库存和做盘点调整。

### 1. 看库存

页面表格里会显示：

- 名称
- 编码
- 规格
- 单位
- 当前库存
- 最近业务日期
- 最近一次更新时间
- 库存依据摘要

### 2. 查找某个产品库存

- 在搜索框输入名称。

### 3. 做盘点或库存调整

1. 找到对应产品。
2. 点“盘点/调整”。
3. 选择调整方式。
4. 填数量。
5. 填业务日期。
6. 有需要就在备注里写明原因。
7. 点“确认保存”。

### 4. 两种调整方式是什么意思

- 实盘覆写：直接把库存改成你盘点后的实际数量
- 增减调整：在原来库存基础上加上或减去一个数量

举例：

- 现在库存是 10，盘点后发现其实是 8，就用“实盘覆写”，填 8
- 现在库存是 10，刚补录了 2 件，就用“增减调整”，填 +2

### 5. 调整前先看预演提示

弹窗里会显示：

- 当前库存
- 事件前库存
- 事件后库存
- 变化数量

请先确认这些数字看起来对不对，再点保存。

## 七、物料/加工库怎么用

左边点“物料/加工库”。

这里是基础资料库，供入库登记时选择。

页面分成两部分：

- 物料库
- 加工库

### 1. 新增物料或加工项

1. 先点上方对应标签。
2. 输入名称。
3. 编码、规格可以不填。
4. 单位尽量填清楚。
5. 默认单价按需要填写。
6. 点“保存”。

### 2. 修改资料

- 在列表中点“编辑”
- 改完后点“保存修改”

### 3. 停用与启用

- 点“停用”后，旧数据不会消失，但以后一般不再继续选用
- 需要恢复时，再点“启用”

### 4. 删除

只有“没有被历史明细使用过”的资料，才可以删除。

如果已经参与过流水，系统会不让删，这是正常的。

## 八、产品管理怎么用

左边点“产品管理”。

这里是销售出库等业务会用到的产品资料库。

### 1. 新增产品

1. 填产品名称。
2. 编码、规格可以按实际情况填写。
3. 填单位。
4. 默认单价按需要填写。
5. 点“保存产品”。

### 2. 看产品状态

列表里能看到：

- 当前库存
- 状态是“启用”还是“停用”
- 是否“可删”

### 3. 编辑产品

- 点“编辑”
- 改好后点“保存修改”

### 4. 停用产品

如果某个产品以后不用了，可以先“停用”，不要急着删除。

这样做更安全，因为旧流水还会保留。

### 5. 删除产品

只有没有参与历史流水的产品，才可以删除。

如果显示“不可删”，就说明它已经被以前的记录使用过。

## 九、用户管理怎么用

只有管理员或最高管理员才能看到左边的“用户管理”。

### 1. 审核新用户注册

进入“用户管理”后，先看“注册申请审核”。

操作步骤：

1. 看申请人的用户名。
2. 确认是否允许使用。
3. 点“通过”或“拒绝”。

### 2. 查看系统已有用户

在“系统用户”区域，可以看到：

- 用户名
- 角色
- 状态
- 创建时间

### 3. 启用或禁用用户

- 点“禁用”后，这个用户不能登录
- 点“启用”后，这个用户可以重新登录

### 4. 重置密码

1. 找到对应用户。
2. 点“重置密码”。
3. 输入新密码。
4. 点“确认重置”。

### 5. 修改用户名

1. 找到对应用户。
2. 点“修改用户名”。
3. 输入新的用户名。
4. 点“确认修改”。

如果修改的是当前正在登录的账号，页面可能会自动刷新，这是正常的。

### 6. 角色说明

- 最高管理员：权限最大
- 日常管理员：能做大部分管理工作
- 普通用户：主要用于日常记账和查看

### 7. 最高管理员要注意

- 最高管理员角色不能随便更改
- 普通管理员不能修改最高管理员的某些信息

## 十、备份与恢复怎么用

这部分通常只有管理员能看到。

位置在首页右上角的“备份与恢复”。

### 1. 什么时候建议备份

- 大量录入前
- 大量修改前
- 删除数据前
- 每周或每月固定备份一次

### 2. 怎么导出备份

1. 点“备份与恢复”。
2. 选好备份范围。
3. 点“导出 JSON 可恢复备份”。
4. 把下载下来的文件保存在安全位置。

### 3. 恢复前一定注意

- 恢复前最好先再备份一次
- 不清楚恢复策略时，不要贸然执行
- 如果是重要数据，建议先找管理员确认

### 4. 恢复时的大致步骤

1. 点“备份与恢复”。
2. 点“导入备份（预检查）”。
3. 先看预检查结果。
4. 确认没有问题后，再点“执行恢复”。

## 十一、手机上怎么操作

系统在手机上也能使用，但屏幕较小，建议注意以下几点：

- 先看左上或左侧能不能展开菜单
- 点菜单时慢一点，避免误触
- 表格内容多时，可以左右滑动查看
- 填写表单时，写完一项再看下一项，不要着急

## 十二、常见问题

### 1. 点了按钮没反应怎么办

- 先等几秒
- 看按钮有没有变成“保存中”“处理中”
- 再看页面有没有弹出提示
- 如果还是不行，再刷新页面重新操作

### 2. 刷新页面会不会把系统数据弄丢

- 一般不会丢系统里已经保存的数据
- 但如果你正在填写表单，还没点“确认登记”或“保存”，那未保存的内容可能会丢

### 3. 记录填错了怎么办

- 如果是刚登记的，可以到相关页面找到记录后删除
- 如果不确定能不能删，先问管理员

### 4. 为什么我看不到某些按钮

- 很可能是因为当前账号权限不够
- 例如导出、备份、用户管理等功能，普通用户可能看不到

### 5. 为什么删不掉产品、物料或加工项

- 因为它已经被历史流水用过了
- 这种情况下建议“停用”，不要强删

### 6. 忘记密码怎么办

- 联系管理员
- 让管理员在“用户管理”里帮你重置密码

## 十三、最简操作顺序

如果你只想记住最常用的顺序，就记下面这几句：

1. 先登录
2. 左边找菜单
3. 记账就点“记账登记”
4. 查客户就点“客户与欠款”
5. 看库存就点“库存管理”
6. 不会删就先别删
7. 点保存后等提示
8. 退出时点“退出登录”

## 十四、给使用人员的提醒

- 宁可慢一点，也不要连续乱点
- 看不懂的提示，先停下来，再问人
- 重要数据操作前，最好先备份
- 录入时最容易错的是日期、金额、数量，请多看一眼
